## 🚀 How to Use

**Step 1:** Download the Excel file above

**Step 2:** Open in Excel or Google Sheets

**Step 3:** For each checklist item:
- Read the item and "Why it matters" column
- Complete the action
- Set status: ✅ Done / ❌ Not Done / ⚠️ In Progress

**Step 4:** Use the color coding to track
- 🟢 Green = Done
- 🟡 Amber = In Progress
- 🔴 Red = Not Done / Blocked
